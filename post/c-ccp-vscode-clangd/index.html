搭建C/Cpp开发环境: Vscode + C/CPP + Clangd

2023-02-13
 | linux C/C++
 | reads

在vscode中安装 C/C++ clangd插件,并在配置clangd插件 然后再去github下载clangd的二进制文件，在插件配置里修改路径 编译采用cmake或者make都是可以的。

    "clangd.onConfigChanged": "restart",
    "clangd.arguments": [
        "--clang-tidy",  // 静态检查
        "--compile-commands-dir=${workspaceFolder}/build",
        "--completion-style=bundled",
        "--enable-config",
        "--fallback-style=Google",  // 编码风格
        "--function-arg-placeholders=false",
        "--header-insertion-decorators",
        "--header-insertion=iwyu",  // 允许补充头文件
        "--log=verbose",  // 让 Clangd 生成更详细的日志
        "--pch-storage=memory",  // pch优化的位置(memory 或 disk，选择memory会增加内存开销，但会提升性能)
        "--pretty",
        "--ranking-model=heuristics", // 建议排序模型
        "-j=12",
        "clangd.checkUpdates": true, //  自动检测 clangd 更新
    ],
    "clangd.fallbackFlags": [
        "-std=c++11",
        "-std=c99"
    ],
    "C_Cpp.intelliSenseEngine": "disabled", // 一定需要设置，不然C/C++ 插件提示会有冲突

目前就是如此配置，后续可能会转战clang + clangd + lldb + cmake,可以参考: https://www.cnblogs.com/lidabo/p/15637226.html (发布日期更早些)进行配置。
